# Giovanni Skills

Skills para Claude Code e agentes compatíveis com o formato `.skill`.

## Estrutura de pastas

```text
giovanni-skills/
  skills/
    cto-cofounder/
      SKILL.md
      references/
    principal-engineer/
      SKILL.md
      references/
```

## Skills

### cto-cofounder
Co-founder técnico (CTO) para decisões estratégicas: arquitetura, stack, segurança, LGPD, integrações, time e custo de infraestrutura.

### principal-engineer
Operacionaliza decisões técnicas: ADRs, padrões de codebase, documentação, onboarding de devs, processo de PR review e roadmap de dívida técnica.

## Características

- Sem dependências externas obrigatórias
- Regras explícitas para segurança, code review, simplificação e testes
- Fronteira clara entre decisão estratégica (CTO) e execução operacional (Principal)
- Estrutura pronta para reutilização em outros projetos
